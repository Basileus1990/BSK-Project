Verifies the digital signature found in a PDF document against a provided public key. 

This function reads a PDF, extracts its first embedded signature, and performs two main checks\+:
\begin{DoxyEnumerate}
\item It compares the public key embedded in the signature\textquotesingle{}s certificate with the {\ttfamily public\+\_\+key} argument. If they do not match, verification fails (returns {\ttfamily False}).
\item It validates the integrity of the signature itself using {\ttfamily pyhanko}\textquotesingle{}s validation mechanism. For self-\/signed certificates, it creates a {\ttfamily Validation\+Context} trusting the embedded certificate itself to validate the signature.
