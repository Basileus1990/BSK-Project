\doxysubsection{Detailed Description}
Provides functionality to verify digital signatures in PDF documents. 

This module uses {\ttfamily pyhanko} and {\ttfamily cryptography} libraries to validate the integrity of a PDF signature and compare the embedded public key with a provided public key.
