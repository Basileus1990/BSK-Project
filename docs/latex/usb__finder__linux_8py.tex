\doxysubsection{Detailed Description}
Provides functions to find USB device mount paths on Linux systems. 

This module uses system utilities like {\ttfamily glob} and {\ttfamily lsblk} to identify connected USB storage devices and determine their mount points. The primary function {\ttfamily get\+\_\+usb\+\_\+mount\+\_\+paths\+\_\+linux} is intended for use by other modules needing to access files on USB drives.
