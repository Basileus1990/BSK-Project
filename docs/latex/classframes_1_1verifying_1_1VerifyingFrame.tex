\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classframes_1_1verifying_1_1VerifyingFrame}{Verifying\+Frame}} class provides the UI for PDF signature verification. 

This frame allows users to select a PDF file and a public key file. It then attempts to verify the PDF\textquotesingle{}s signature using the provided key and displays the result (valid, invalid, or error messages). Inherits from tk.\+Frame.
